# Youtube Lite

Live URL: https://video-player-teal-six.vercel.app/

## Getting started
```  
git clone https://github.com/shreeyansh-29/Video-Player
cd Video-Player
yarn install  
```  
Running the below command will start the application  
```  
yarn run dev
#(Local application will run on port 5173)
